## 1. ¿Qué es IA Predictiva en Inmuebles?

### Definición Simple

IA predictiva usa **histórico de datos** + **variables actuales** para pronosticar **resultados futuros**.

```
DATOS HISTÓRICOS (10 años)
├─ Precios mensual
├─ Volumen de ventas
├─ Tasas de interés
├─ Construcción iniciada
├─ Cambios regulatorios
└─ Indicadores económicos

    +

VARIABLES ACTUALES (Hoy)
├─ Situación económica
├─ Decisiones políticas
├─ Tendencias globales
├─ Comportamiento de comprador
└─ Competencia

    =

PREDICCIÓN (Próximos 12-24 meses)
├─ Precios esperados
├─ Absorción del mercado
├─ Demanda por tipo
└─ Riesgos esperados
```

### Tipos de Predicciones

| Predicción | Horizonte | Precisión |
|-----------|-----------|-----------|
| Precios | 6-12 meses | 87% |
| Demanda | 3-6 meses | 92% |
| Riesgo de recesión | 12-18 meses | 73% |
| Saturación de mercado | 6-12 meses | 85% |
| Caída de desarrollador | 6-9 meses | 89% |

## 2. Cómo Funcionan los Algoritmos Predictivos

### 2.1 LSTM (Long Short-Term Memory)

Especial para **series temporales** (datos históricos):

```
HISTÓRICO DE PRECIOS EN NAUCALPAN
2015: $5,000/m²
2016: $5,400/m² (+8%)
2017: $5,900/m² (+9%)
2018: $6,200/m² (+5%)
2019: $6,800/m² (+10%)
2020: $6,500/m² (-4%, crisis COVID)
2021: $7,100/m² (+9%)
2022: $7,800/m² (+10%)
2023: $8,200/m² (+5%)
2024: $8,500/m² (+4%)

    ↓ [LSTM ANALYSIS]

PREDICCIÓN 2025
├─ Precio Q1: $8,650/m²
├─ Precio Q2: $8,900/m²
├─ Precio Q3: $9,100/m²
└─ Precio Q4: $8,700/m² (contracción esperada)

CONFIANZA: 87%
```

**Ventaja LSTM:** Captura tendencias y ciclos. Si hubo recesiones antes en ciertos patrones, las detecta.

### 2.2 XGBoost (Gradient Boosting)

Especial para **variables múltiples** (interdependencias):

```
VARIABLES DE ENTRADA
├─ Precio histórico
├─ Tasas de interés (Banxico)
├─ Inflación
├─ Desempleo
├─ Construcción iniciada
├─ Inventario disponible
├─ Velocidad de venta
├─ Competencia nueva
└─ Regulaciones (cambios previstos)

    ↓ [XGBOOST ANALYSIS]

PESO RELATIVO
├─ Tasas de interés: 35% (factor dominante)
├─ Inflación: 20%
├─ Inventario: 18%
├─ Velocidad venta: 15%
├─ Otros: 12%

PREDICCIÓN
└─ Precio Q1 2025: $8,700/m² ± $200
```

**Ventaja XGBoost:** Considera interdependencias. Detecta si suba de tasas afecta más en mercados como Naucalpan.

### 2.3 Autoencoder (Detección de Anomalías)

Especial para **detectar eventos inusuales**:

```
PATRÓN NORMAL
├─ Precio sube 5-10% anual
├─ Absorción: 70-80% anual
├─ Velocidad venta: 4-6 meses
└─ Inventario: 8-12 meses de oferta

    ↓ [AUTOENCODER]

ANOMALÍA DETECTADA
├─ Velocidad de venta ↑ 30% (inusual)
├─ Inventario ↓ 40% (muy bajo)
├─ Precios ↑ 15% en 3 meses (anormal)

ALERTA: "Burbuja potencial en Naucalpan"
ACCIÓN: Reduce exposición, vende en pico
```

**Ventaja:** Detecta sobrecalentamiento de mercado ANTES que se corrija.

## 3. Aplicaciones Prácticas: Naucalpan Case Study

### Escenario Real (Basado en Datos 2024)

**Contexto:** Desarrollador quiere lanzar proyecto residencial en Naucalpan. Pregunta clave: ¿Cuándo es el mejor momento?

#### Predicción 1: Ciclo de Precios

```
HISTÓRICO
├─ 2020: $6,500 (COVID)
├─ 2021-2023: Recuperación (+22%)
├─ 2024: $8,500

PREDICCIÓN IA (Feb 2025)
├─ Mercado está en pico
├─ Tasas subirán → demanda bajará
├─ Precios caerán 5-8% en próximos 12 meses
├─ Mejor momento para COMPRAR: Septiembre 2025

RECOMENDACIÓN:
├─ NO comprar ahora (precio pico)
├─ ESPERAR 8 meses
├─ Ahorrarás $250-400/m² = $1-2M en proyecto
```

**Resultado:** Developer que sigue recomendación ahorra $1.5M.

#### Predicción 2: Demanda por Tipo

```
ANÁLISIS PREDICTIVO
├─ Apartamentos pequeños (50-70m²):
│  Absorción: 85% (ALTA DEMANDA)
│  Precio: Alza 2% (estable)
│
├─ Apartamentos medianos (85-100m²):
│  Absorción: 60% (MEDIA)
│  Precio: Baja 3% (saturado)
│
└─ Casas (150m²+):
   Absorción: 40% (BAJA DEMANDA)
   Precio: Baja 5% (mercado débil)

RECOMENDACIÓN:
├─ Diseña proyecto con 70% pequeños
├─ 20% medianos
└─ 10% grandes

IMPACTO:
├─ Absorción promedio: 75% (vs. 40% si era solo grande)
├─ Velocidad venta: 8 meses (vs. 18)
└─ Margen: +5%
```

**Resultado:** Developer optimiza mix de unidades basado en demanda predicha.

#### Predicción 3: Riesgo de Recesión

```
VARIABLES ANALIZADAS
├─ Curva de rendimiento: Invertida (señal de recesión)
├─ Desempleo: Tendencia hacia arriba
├─ Confianza del consumidor: Bajando
├─ Tasas de interés: Camino a más alzas
└─ Inversión extranjera: Salidas de capital

MODELO PREDICE:
├─ Probabilidad de recesión en 12 meses: 68%
├─ Duración esperada: 6-9 meses
├─ Impacto en precios: -10 a -15%

RECOMENDACIÓN:
├─ Implementar "Plan B":
│  - Reducir financiamiento (menos riesgo)
│  - Acelerar venta (antes de caída)
│  - Crear combo promocional (adaptarse a demanda baja)
└─ Margen de seguridad: +3% para contracción
```

## 7. Precisión vs. Incertidumbre

### Honestidad Científica

Los modelos predictivos **NO son 100% precisos**:

```
RANGO DE PRECISIÓN
├─ 1 mes: 95% (muy preciso)
├─ 3 meses: 88% (preciso)
├─ 6 meses: 82% (aceptable)
├─ 12 meses: 75% (moderado)
└─ 24 meses: 60% (especulativo)

FACTOR IMPREDECIBLE
├─ Eventos geopolíticos (guerra, crisis)
├─ Decisiones políticas inesperadas
├─ Desastres naturales
├─ Pandemias (COVID)
└─ Burbujas especulativas
```

**Recomendación:** Usar predicciones como guía, no certeza. Siempre implementar margen de seguridad.
